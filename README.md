# UPnP Port Mapper GUI

Python と `upnpy` を使って、UPnP 対応ルーターのポート開放を GUI で行うツールです。  
`tkinter` ベースなので、コマンドライン操作に慣れていなくても扱いやすい構成にしています。

## 特徴

- UPnP 対応ルーターを自動検索
- 利用可能なルーター / サービス候補を選択可能
- グローバル IP を表示
- 外部ポート / 内部ポート / プロトコル / 内部 IP / 説明を指定してポート開放
- 既存のポートマッピング一覧を取得して表示
- 一覧をクリックしてフォームに反映
- 入力値チェック付き
- ログ表示あり

## 動作環境

- Python 3.10 以上推奨
- Windows での利用を想定
- 同一 LAN 内にある UPnP 対応ルーター
- ルーター側で UPnP が有効になっていること

## 依存ライブラリ

このプロジェクトで追加インストールが必要なのは主に `upnpy` です。

```bash
pip install upnpy
```

`tkinter` は多くの Python 環境で標準搭載されています。

## 使い方

1. このリポジトリを取得します。
2. 必要であれば仮想環境を作成します。
3. `upnpy` をインストールします。
4. 以下のコマンドで起動します。

```bash
python upnp_gui.py
```

## 操作手順

1. 起動後、自動で UPnP デバイスの検索が始まります。
2. 候補が複数ある場合は対象のルーターを選びます。
3. `外部ポート`、`内部ポート`、`プロトコル`、`内部 IP`、`説明` を入力します。
4. `ポートを開放 / 更新` を押すとポートマッピングを追加します。
5. `一覧を更新` で現在のマッピング一覧を再取得できます。
6. 一覧から既存設定を選ぶとフォームに内容が反映されます。
7. `フォーム内容で削除` を押すと、指定した外部ポートとプロトコルの設定を削除します。

## 注意事項

- UPnP は便利ですが、セキュリティ上のリスクもあります。不要なポートは開放しっぱなしにしないでください。
- ルーターの実装によっては、一部の UPnP 機能が使えないことがあります。
- 既に別の端末やアプリが使っているポートは、競合のため追加できない場合があります。
- セキュリティソフトやルーター設定によっては、UPnP の検索や追加が失敗することがあります。
